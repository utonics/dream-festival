# 꿈-드림 페스티벌 미래기술존 운영 관리 시스템

## 개요
2026 영천시 청소년 진로진학 박람회 「꿈-드림(Dream) 페스티벌」 미래기술존(5개 존, 15개 프로그램)의 운영 전반을 관리하는 통합 시스템이다. 관리자(운영 스태프)와 참가자(청소년) 양쪽 모두를 위한 듀얼 인터페이스를 제공하며, 실시간 현장 상황 파악과 체험 게이미피케이션을 핵심 축으로 삼는다.

## 디자인 방향

**톤앤매너**: "청소년 과학관의 안내 키오스크" — 밝고 에너제틱하되 정보 밀도가 높은 실용적 인터페이스. 어둡거나 무거운 대시보드 느낌이 아니라, 박람회 현장에서 서서 쓸 수 있을 만큼 직관적이고 경쾌한 톤.

**색감**: 영천시 시 상징색(녹색 계열)을 베이스로, 각 존별로 구분되는 5색 액센트 컬러 시스템. A존-코랄, B존-앰버, C존-시안, D존-라임, E존-라벤더. 배경은 밝은 오프화이트 + 미세한 도트 그리드 패턴.

**명시적 금지**: 보라색-핑크 그라데이션 히어로 배너, 둥근 모서리 흰색 카드 나열, 아이콘+제목+설명 3열 반복 그리드. 이런 AI slop 패턴 절대 사용하지 않는다.

**차별화 아이디어 1 — "관제탑 뷰"**: 관리자 대시보드의 메인 화면은 미래기술존 5개 존의 물리적 배치를 반영한 평면도(floorplan) 형태로 구성. 각 부스가 지도 위의 노드로 표시되고, 실시간 혼잡도/운영상태가 노드 색상과 펄스 애니메이션으로 나타남.

**차별화 아이디어 2 — "스탬프 콜렉션 보드"**: 참가자 화면은 15개 프로그램을 5x3 허니콤(육각형) 그리드로 배치한 스탬프 보드. 미완료는 흐린 실루엣, 완료 시 존 컬러로 채워지며 마이크로 파티클 이펙트 발생. 전체 완성 시 특별 컴플리션 애니메이션.

## 기능 목록

### 기능 1: 존 플로어맵 실시간 대시보드 (관리자)
- 설명: 5개 존 × 15개 부스를 물리적 배치 기반 평면도로 시각화하는 관제 화면
- 사용자 스토리: 운영 본부장이 한 화면에서 전체 미래기술존의 현재 상태를 파악하고, 문제 있는 부스를 즉시 식별할 수 있다
- 포함 요소:
  - 존별 색상 코딩된 평면도 레이아웃 (A~E존 5색)
  - 각 부스 노드에 실시간 상태 표시 (운영중/대기/점검/종료)
  - 부스별 현재 체험 인원 수 + 대기열 수 표시
  - 혼잡도에 따른 노드 크기/펄스 변화 (녹색=여유, 노란색=보통, 빨간색=혼잡)
  - 부스 노드 클릭 시 상세 패널 슬라이드 오픈 (해당 프로그램 정보, 스태프, 통계)
  - 전체 참가자 수, 현재 체험 중 인원, 스탬프 완료율 등 상단 요약 지표

### 기능 2: 프로그램 일정 타임테이블 (관리자)
- 설명: 15개 프로그램의 시간대별 운영 스케줄을 간트 차트 형태로 관리
- 사용자 스토리: 운영 담당자가 각 프로그램의 시작/종료 시간, 세션 구분, 휴식 시간을 설정하고 당일 일정 변경을 실시간 반영할 수 있다
- 포함 요소:
  - 시간축(가로) × 프로그램(세로) 간트 차트 뷰
  - 드래그로 세션 시간 조정 가능
  - 존별 그룹핑 + 컬러 바
  - 현재 시간 인디케이터 라인
  - 점심시간/정비시간 등 블록 타임 설정
  - 일정 변경 시 참가자 화면 자동 반영

### 기능 3: 스태프 배치 관리 (관리자)
- 설명: 각 부스별 운영 인력 배치 현황을 관리하고 즉석 재배치 지원
- 사용자 스토리: 인력 관리자가 스태프 명단을 존/부스에 배정하고, 결원 발생 시 빠르게 대체 인력을 지정할 수 있다
- 포함 요소:
  - 스태프 목록 (이름, 역할, 배정 존/부스, 연락처)
  - 드래그 앤 드롭으로 부스 간 인력 이동
  - 부스별 필요 인원 vs 현재 배치 인원 비교 표시
  - 미배정 스태프 풀 사이드바
  - 역할 태그 (운영, 기술지원, 안내, 멘토)

### 기능 4: 참가자 통계 & 분석 (관리자)
- 설명: 존별/시간대별/프로그램별 참가자 데이터를 시각화하는 분석 화면
- 사용자 스토리: 운영진이 어떤 존이 가장 인기 있는지, 어떤 시간대에 몰리는지를 파악하여 현장 운영 판단에 활용할 수 있다
- 포함 요소:
  - 존별 참가자 수 도넛 차트 (5색 매칭)
  - 시간대별 참여 추이 영역 차트
  - 프로그램별 체험 완료 수 수평 바 차트
  - 스탬프 투어 진행률 분포 (0~15개 완료 히스토그램)
  - 실시간 갱신 토글 (자동/수동)
  - AI 인사이트 패널: "14시대 C존 혼잡 예상" 등 간단한 예측 문구

### 기능 5: 콘텐츠 등록 & 관리 (관리자)
- 설명: 15개 프로그램의 상세 정보를 CRUD로 관리
- 사용자 스토리: 콘텐츠 담당자가 각 프로그램의 제목, 설명, 썸네일, 난이도, 소요시간, 최대 인원 등을 등록/수정할 수 있다
- 포함 요소:
  - 프로그램별 편집 폼 (제목, 설명, 존 분류, 아이콘, 난이도 1~3, 소요시간, 정원)
  - 썸네일 이미지 업로드/미리보기
  - 프로그램 활성/비활성 토글
  - 존별 필터 + 검색
  - 변경 이력 로그

### 기능 6: 허니콤 스탬프 투어 보드 (참가자)
- 설명: 15개 프로그램을 5×3 육각형 허니콤 그리드로 배치한 디지털 스탬프 수집 화면
- 사용자 스토리: 참가 청소년이 자신의 스탬프 보드를 보며 어떤 존을 방문했는지 한눈에 파악하고, 다음에 갈 부스를 정할 수 있다
- 포함 요소:
  - 5×3 육각형 허니콤 그리드 (존별 컬러 그룹핑)
  - 미방문 부스: 반투명 실루엣 + 프로그램 아이콘
  - 방문 완료 부스: 존 컬러로 채워짐 + 체험 날짜/시간 표시
  - 스탬프 획득 시 파티클 축하 애니메이션
  - 전체 15개 완료 시 "꿈-드림 마스터" 컴플리션 애니메이션 + 인증서 화면
  - 진행률 프로그레스 바 (X/15)
  - 허니콤 셀 탭 시 해당 프로그램 상세 정보 팝업

### 기능 7: QR 기반 체험 체크인/결과물 다운로드 (참가자)
- 설명: QR코드 스캔으로 부스 체험을 기록하고, 체험 결과물을 다운로드하는 기능
- 사용자 스토리: 참가자가 부스에 도착하여 QR을 스캔하면 체크인되고, 체험 완료 후 자신의 결과물(아바타, 분석 리포트 등)을 QR로 받을 수 있다
- 포함 요소:
  - QR 스캐너 화면 (카메라 활성화 + 가이드 프레임)
  - 체크인 성공 시 부스 정보 + 스탬프 획득 확인 화면
  - 체험 결과물 목록 (다운로드 가능한 파일/이미지)
  - 결과물 QR코드 생성 (다운로드 링크)
  - 체크인 이력 타임라인

### 기능 8: 체험 랭킹 리더보드 (참가자)
- 설명: 댄스 챌린지, AI 분류 대회 등 경쟁형 프로그램의 실시간 랭킹 표시
- 사용자 스토리: 참가자가 자신의 점수와 순위를 확인하고, 다른 참가자와 경쟁하며 재도전 동기를 얻을 수 있다
- 포함 요소:
  - 프로그램별 탭 전환 (댄스 AI 챌린지, AI 분류 챌린지, 프롬프트 아트 챌린지)
  - 상위 10명 하이라이트 + 본인 순위 강조
  - 점수/기록 기반 정렬
  - 최근 기록 갱신 시 "NEW" 뱃지 애니메이션
  - 참가자 닉네임 + 존 컬러 아바타 아이콘

### 기능 9: 실시간 부스 혼잡도 맵 (참가자)
- 설명: 참가자가 각 부스의 현재 대기 상황을 확인할 수 있는 시각적 맵
- 사용자 스토리: 참가 청소년이 다음에 어디를 갈지 정할 때, 지금 비어 있는 부스를 빠르게 찾을 수 있다
- 포함 요소:
  - 5개 존 간소화 맵 (관리자 플로어맵의 참가자 버전)
  - 부스별 혼잡도 3단계 색상 표시 (여유/보통/혼잡)
  - 예상 대기 시간 표시
  - AI 추천: "지금 가기 좋은 부스" 상위 3개 자동 추천
  - 부스 탭 시 프로그램 정보 + 위치 안내

### 기능 10: 참가자 프로필 & 체험 이력 (참가자)
- 설명: 참가자 개인 프로필과 전체 체험 기록을 관리하는 마이페이지
- 사용자 스토리: 참가자가 자신의 닉네임을 설정하고, 오늘 하루 어떤 체험을 했는지 타임라인으로 돌아볼 수 있다
- 포함 요소:
  - 닉네임/아바타 설정 (사전등록 or 현장등록)
  - 체험 타임라인 (시간순 카드 스트림)
  - 획득 스탬프 수 / 총 체험 시간 / 참여 존 수 요약
  - 체험 결과물 갤러리 (아바타, 이미지, 리포트 모아보기)
  - 참가 인증서 PDF 생성 버튼 (이름 + 스탬프 수 + 날짜)

### 기능 11: AI 스마트 동선 추천 (참가자)
- 설명: 참가자의 관심사와 현재 혼잡도를 조합하여 최적 체험 순서를 추천하는 AI 기능
- 사용자 스토리: 처음 온 참가자가 "나에게 맞는 코스"를 받아 효율적으로 박람회를 돌 수 있다
- 포함 요소:
  - 간단 관심사 설문 (3문항: 좋아하는 분야, 체력 수준, 남은 시간)
  - AI가 추천하는 5~8개 프로그램 순서 코스
  - 각 추천 이유 한 줄 설명 ("모션에 관심 있으니 A존부터!")
  - 현재 혼잡도 반영한 동적 재추천
  - 추천 코스를 스탬프 보드에 하이라이트 표시

### 기능 12: 알림 & 공지 시스템
- 설명: 운영진이 참가자에게 실시간 공지를 보내고, 이벤트 알림을 자동 발송하는 시스템
- 사용자 스토리: 운영진이 "C존 프롬프트 아트 챌린지 15분 후 마감!" 같은 긴급 공지를 전체 또는 특정 존 참가자에게 보낼 수 있다
- 포함 요소:
  - 관리자: 공지 작성 (전체/존별/프로그램별 대상 선택)
  - 관리자: 예약 공지 설정
  - 참가자: 알림 배너 + 알림 목록 페이지
  - 자동 알림 트리거 (랭킹 변동, 스탬프 달성, 이벤트 시작 5분 전)
  - 공지 이력 관리
